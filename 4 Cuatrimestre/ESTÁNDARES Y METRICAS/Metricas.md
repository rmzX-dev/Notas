
Indirectas  

- HPT (horas-programador totales)
- LCFH (líneas de código fuente por hora de programación )
    LCFH = LCF/HPT
- CPT (Costo total actual del proyecto en unidades monetarias)
	CTP = CHP * HPT
- CLCF (Coste por linea de código fuente )
	 CLCF = LCF /CTP
